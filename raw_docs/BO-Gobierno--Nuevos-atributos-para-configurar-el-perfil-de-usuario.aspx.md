Resumen: 
Nos han solicitado nuevas configuraciones de usuarios del BO Gobierno, para limitar o filtrar la visualización de datos de ciertos usuarios. 

Las nuevas dimensiones a implementar son las siguientes 
 Filtro por informes 
Se deberá poder establecer a que conjunto de los informes presentes en la plataforma, tendrá acceso cada usuario configurado. 
 Filtro por proyecto especial 
Se debe poder definir si a un usuario se le presentan solamente la información perteneciente a un proyecto especial (como es el caso del proyecto cooling2feed).
