 Consulta del cdigo de verificacin / recogida ***REVISAR dinamismo a partir de _DOM.cua_master*** 

 Este cdigo corresponde al parmetro " eatc-verification_code ", del respectivo encabezado de donacin ( eatc_dona_headers ). 
 {{URL_entorno_donantes}}/api/{{_DOM. cua_master }}/eatc_dona_headers?eatc-id={{valor}}  

 Ejemplo: 
 El para el anuncio cuyo " eatc-id " = 8687012  

 Ambiente productivo: https://donantes.eatcloud.info/api/abaco/eatc_dona_headers?eatc-id=8687012   

 El cdigo de verificacin (eatc-verification_code) es: 715518    
 Este cdigo se le muestra en pantalla al usuario, para que lo entregue a la hora de la llegada al punto de donain y el mismo sea validado mediante la funcionalidad " Entrega de donacin: hora de llegada (eatc_doma_checkin) " 

 En etapas posteriores se podr generar el cdigo QR correspondiente a eatc-verification_code para facilitar el proceso de verificacin en el punto (que no se tenga que digitar el cdigo).
